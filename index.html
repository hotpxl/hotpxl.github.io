Athena

  • Last Active: December, 2014
  • Repository: Coming Soon!

  • Description:

    Make deep neural networks a home dish! This is a project I am participating at Microsoft Research Asia. Our goal is to provide an intuitive tool aimed at deep neural networks, that is optimized for CUDA. It has a Python interactive frontend. Of course, precompiled code is also supported. On the backend side, it utilizes dataflow to achieve maximum parallelism and performance, on multiple GPU/CPU configuration.

    I worked mainly on the backend, including but not limited to

    • Directed acyclic graph manipulation
    • Memory management
    • Thread management
    • Device management
    • Dispatching routine

FTP Server and Client

  • Last Active: May, 2013
  • Repository: GitHub

  • Description:

    It is a class project for Computer Networks. I implemented it in Python 3.3 and strictly according to RFC 959. Though it does not offer a GUI, it supports the most essential, but not all, features of the RFC. It even supports different network layers, including IPv4 and IPv6. Also the server has support for multitasking, allowing it to respond to different clients at the same time.
